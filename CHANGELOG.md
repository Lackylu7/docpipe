# Changelog

## 0.1.0

Initial product MVP for local enterprise document preprocessing.

### Added

- Local Streamlit workbench with Chinese and English language support
- CLI conversion, demo, history, engine registry, and workflow template commands
- Parser routing across production adapters with planned adapter slots
- Markdown, JSON, RAG chunk, and vendor starter exports
- Review checklist, handoff guide, manifest, and export ZIP
- Workflow templates for general, enterprise policy, support FAQ, and product manual use cases
- Included English and Chinese sample documents
- Docker and Windows launcher scripts
- Automated test suite and GitHub Actions CI
