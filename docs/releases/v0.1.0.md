# DocPipe v0.1.0 Release Draft

DocPipe v0.1.0 is the first public MVP for local enterprise document preprocessing.

## Highlights

- Local Web workbench
- CLI conversion and demo commands
- Chinese and English UI/output
- Workflow templates for policy, support FAQ, product manual, and general use
- Markdown, JSON, RAG chunks, review checklist, handoff guide, and export ZIP
- Starter exports for Dify, Coze, FastGPT, RAGFlow, and custom RAG systems
- GitHub Actions CI
- Included English and Chinese samples
- Synthetic company knowledge-base case study

## Good Fit

- local demos
- internal pilots
- service-led document cleanup
- knowledge-base import preparation

## Not Yet Included

- user accounts
- background job queue
- production SaaS billing
- enterprise SSO
- one-click vendor imports
- reverse Markdown-to-Word/PDF/PPT export

## Suggested Release Notes

This release is suitable for experimentation and pilot projects. For customer use, review
`docs/limitations.md` and run a validation pass with real documents before import.
