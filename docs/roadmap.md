# Roadmap

DocPipe should stay focused on one core promise:

> Turn messy enterprise documents into reviewable knowledge-base import packs.

## Current MVP

- Local Web workbench
- CLI batch conversion
- Chinese and English interface/output
- Workflow templates
- Synthetic mixed-company load test
- Review checklist and handoff guide
- Starter exports for common knowledge-base workflows

## Next: 0.2

- Real PDF, Word, Excel, and PowerPoint validation set
- One-click ZIP download from every successful Web run
- Release package with versioned artifacts
- Basic performance benchmark on larger folders
- More robust PDF/table validation checklist

## Later: 0.3

- Project/workspace management
- Background job queue
- Retry failed files from history
- Saved import profiles for Dify, FastGPT, RAGFlow, and Coze
- Optional authentication for private team deployments

## Not Now

- Markdown-to-Word/PDF/PPT reverse conversion
- Full SaaS billing
- Cloud file storage
- Custom OCR model training
- Automatic production import into every vendor platform
